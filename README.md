# Wishy — Список Желаний

**Wishy** — это веб-приложение, которое позволяет пользователю создавать и отслеживать список желаемых вещей. В нем можно добавлять, удалять и помечать желания как выполненные. Приложение также включает рекомендации песен из Spotify и возможность переключать темы оформления.

## Основной функционал:

- **Добавление желаемых вещей**  
  Пользователь может добавить новое желание в список.

- **Удаление желаемых вещей**  
  Каждое желание можно удалить по кнопке рядом с ним.

- **Отметка выполненных желаний**  
  При клике на элемент списка желание помечается как выполненное.

- **Статистика и прогресс**  
  Отображается количество всех желаемых вещей и количество выполненных. Также имеется визуальный прогресс выполнения через прогресс-бар.

- **Рекомендация песни**  
  По нажатию кнопки "Рекомендовать песню" приложение выводит случайную песню из заранее заданного списка на Spotify.

- **Темная/светлая тема**  
  Пользователь может переключать тему оформления. Выбор сохраняется в `localStorage`, и при перезагрузке страницы тема будет восстановлена.

## Технологии:

- **HTML** для структуры страницы
- **CSS** для стилизации интерфейса
- **JavaScript (ES6)** для реализации логики
- **Fetch API** для взаимодействия с Spotify API
- **localStorage** для сохранения данных между сессиями
- **Web Audio API** для создания звуковых эффектов

## Как запустить проект:

1. Скачайте проект на свой компьютер.
2. Откройте файл `index.html` в любом браузере.
3. Приложение будет работать без необходимости установки сервера или сборки.

## Дополнительно:

- Все данные сохраняются в `localStorage`, что позволяет не потерять их при перезагрузке страницы.
- Рекомендация случайной песни извлекается из заранее заданного списка с помощью API Spotify.
